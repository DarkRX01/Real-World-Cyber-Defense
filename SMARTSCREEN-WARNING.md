## About The Warning

Windows shows SmartScreen warnings for:
- New applications (not yet known to Windows)
- Applications without a digital signature
- Any software from unknown publishers

**This is a safety feature, not a security issue.**

Popular applications you trust (before they were famous) showed the same warning.
